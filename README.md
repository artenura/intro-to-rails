# intro-to-rails

## Project Overview
This is a practice project aimed at learning the basics of Ruby on Rails framework.

## Features
- Basic CRUD operations
- User authentication
- RESTful routes

## Technologies Used
- Ruby on Rails
- PostgreSQL
- Bootstrap

## Installation
To install this project, clone the repository and run:

```
bundle install
rails db:create
rails db:migrate
rails server
```
